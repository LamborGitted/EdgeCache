# EdgeCache

基于 **Cloudflare Pages/Workers \+ KV** 开发的临时存储方案, 简单易用

## 适用场景

临时存储单文件, 实现快速上传与下载, 适合临时分享

## 前置准备

1. 拥有 Cloudflare 账号

2. 准备好项目核心文件：`_worker.js`

## 部署步骤

### 第一步：创建 Cloudflare KV

1. 登录 Cloudflare

2. 切换到 **KV** (找不到可以搜索)

3. **创建KV**(命名随意)
